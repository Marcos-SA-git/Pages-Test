This Web was created using the [HUGO](https://gohugo.io/) framework and the [blowfish](https://blowfish.page/) theme. I added, with the assistance of ChatGPT, some features that the theme lacked, like the zoom and pan characteristic for the mermaid schemes, as well as other layout personalizations so the web looked as I liked to look.

First I was going to use MKDocs, but seen the trajectory of the project, the low maintenance and the lack of some other features used in this web, I decided to learn to use HUGO, despite the fact that it has an steeper learning curve.

The amount of features HUGO has, has allowed me to display my documentation in a better way, not only esthetically, but also navigably.
