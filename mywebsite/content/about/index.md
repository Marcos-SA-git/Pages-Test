Esta web se creó utilizando el framework [HUGO](https://gohugo.io/) y el tema [Blowfish](https://blowfish.page/). Con la ayuda de ChatGPT, añadí algunas funciones que le faltaban al tema, como el zoom y el desplazamiento para los esquemas Mermaid, así como otras personalizaciones de diseño para que la web tuviera el aspecto que yo deseaba.

Inicialmente iba a usar MKDocs, pero viendo la trayectoria del proyecto, su bajo mantenimiento y la falta de algunas funciones utilizadas en esta web, decidí aprender a usar HUGO, a pesar de que su curva de aprendizaje es más pronunciada.

La cantidad de funciones que ofrece HUGO me ha permitido mostrar mi documentación de una forma mejor, no solo estéticamente, sino también en cuanto a la navegación.

Un ejemplo de esto es el sistema dinámico de taxonomías y la posibilidad de crear plantillas personalizadas con las que presentar mi contenido además de una integración súper sencilla para la creación de artículos en múltiples de idiomas. Esto me ha permitido crear y mostrar el contenido de una manera no solo única sino también muy ordenada y profesional.
